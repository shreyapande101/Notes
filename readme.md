#  Create Your Notes 

This is a simple **Notes Web Application** made using **HTML, CSS, and JavaScript**.  
It allows users to write, save, and delete notes easily in a clean interface.


---

 Features

- Create and write notes
- Delete notes easily
- Beginner-friendly project

---
  Language Used

- **HTML** – for structure  
- **CSS** – for styling and gradient design  
- **JavaScript** – for creating and deleting notes  

---

 How to Use

1. Click on the **Create Notes** button  
2. Write your notes in the text area  
3. Notes can be edited anytime  
4. Click the **Delete** icon to remove a note
